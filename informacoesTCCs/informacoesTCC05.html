TCC: Os jogos pedagógicos como ferramentas de ensino

Título: Os jogos pedagógicos como ferramentas de ensino
Nome completo do aluno: Nicole Fonseca
Área: Programação de Sistemas
Instituição de ensino: UNICAMP
Ano de publicação: 2018
Nível de ensino: Pós-doutorado
Resumo: A falta de motivação é a principal causa do desinteresse dos alunos, quase sempre acarretada pela metodologia utilizada pelo professor ao repassar os conteúdos. Para despertar o interesse do aluno para a aprendizagem é necessário o uso de uma linguagem atraente, capaz de aproximá-lo o máximo possível da realidade, transformando os conteúdos em vivência. Portanto, o objetivo deste trabalho é contribuir para com a educação, demonstrando que os jogos podem colaborar no processo de ensino.
